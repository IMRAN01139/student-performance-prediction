# student-performance-prediction
Built a machine learning model using Logistic Regression to predict student performance (pass/fail) based on study time and number of courses. Performed data analysis and visualization, achieving accurate predictions.
# Student Performance Prediction

# Objective
To predict whether a student will pass or fail based on study time and number of courses.

#  Tools & Technologies
- Python
- Pandas
- Matplotlib
- Scikit-learn

#  Features
- Data Analysis
- Data Visualization
- Machine Learning Model (Logistic Regression)

#  Result
The model predicts student performance with good accuracy.

2. Run all cells
